## Example
--------
**Python:**
```py
## hmtai
import hmtai

## Get pic from v1 library
print(hmtai.useHM(version="v1", variant="neko"))
## Get pic from v2 library [Some categories has been renamed]
print(hmtai.useHM(version="v2", variant="sfwneko"))
#and etc.
```
